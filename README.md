# PasteImageToMarkdown
- 把粘贴的图片上传到七牛云，然后把图片链接以markdown的格式插入到编辑器中。
- 安装完成后，请前往Setting->Other Settings->PasteImageIntoMarkdown填写七牛相关信息。

## 保存图片在本地
